## 三、核心场景功能测试（5个）

### 场景1：电商主图（图片输入）
- **输入**：一张蔡氏福宁冰感眼罩电商主图（图片）
- **步骤1**：定性 — 电商主图/首图，🔴最严
- **步骤2**：提取与核实 — 双重读取+关键数据确认
- **步骤3**：广告法审核（涉医暗示/绝对化词/可暗示度按电商标准）
- **验收**：风险分级正确（电商主图=🔴尺度），修改建议对标原文字数

### 场景2：包装背面（含企标对照）
- **输入**：一张蔡氏福宁蒸汽眼罩包装背面（图片）
- **步骤1**：定性 — 包装，🔴最严，触发企标
- **步骤2**：提取与核实 — 含成分/规格/执行标准号等关键数据
- **步骤3**：维度A广告法 + 维度B标签合规 + 维度C企标对照
- **验收**：企标对照已执行或已注明跳过

### 场景3：线下展架文案（文字输入）
- **输入**：线下展架文字文案
- **步骤1**：定性 — 线下展架，🟡中等，不触发企标
- **步骤2**：提取与核实 — 文字无需双重读取
- **步骤3**：广告法审核，尺度自动继承🟡中等
- **验收**：可暗示度不再重复询问，约束确认环节触发（如有中风险）

### 场景4：法务词汇自动匹配
- **输入**：含"轻养""本草养护"等词的文案
- **步骤3a**：读取企微白名单 → 命中词汇标注"已背书"
- **验收**：命中词汇在报告中列出+注明背书依据，但后续维度A照常审核

### 场景5：审核人放行 + 知识沉淀
- **输入**：含审核人认为可放行的新词汇
- **步骤4b**：询问审核人是否录入白名单 → 确认 → 企微文档写入
- **验收**：写入成功（errcode=0），词汇已追加到白名单
