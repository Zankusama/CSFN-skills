# 打假风险判断框架

> 本文档由 `SKILL.md` 步骤3b 按需加载。用于辅助判断文案风险等级。

---

## 核心公式

```
风险等级 = 宣称强度 × 证据可获取性 × 监管敏感度
```

## 三步自检法

**Step 1：标红排查**
- 每个形容词/副词都要过一遍：最、很、非常、绝对...
- 极限词：第一、唯一、国家级、顶级...

**Step 2：证据链核查**
```
功效宣称 → 有检测报告吗？
数据宣称 → 有第三方认证吗？
对比宣称 → 有法律依据吗？
```

**Step 3：降级替换**
| 高风险词 | 安全替换 |
|---------|---------|
| "第一" | "领先"（需有依据） |
| "100%有效" | "有助于改善" |
| "治疗XX" | "适用于XX"（仅描述适用场景） |

## 职业打假人主要攻击点

1. **绝对化用语** — 最容易被抓的"硬伤"
2. **功效宣称无依据** — 特别是涉及医疗效果的表述
3. **数据不实/无法验证** — 百分比、对比数据等
4. **极限词** — "第一"、"唯一"、"国家级"等

## 日化品类特有禁忌

| 禁忌类型 | 典型词汇 | 备注 |
|---------|---------|------|
| 化妆品禁用 | 药用、治疗、抗菌 | 除非有特殊化妆品批号 |
| 普通日用品禁用 | 消毒、杀菌 | 需要卫消证字 |
| 医疗暗示禁用 | 消炎、镇痛、活血化瘀 | 日用品不能有医疗效果承诺 |
