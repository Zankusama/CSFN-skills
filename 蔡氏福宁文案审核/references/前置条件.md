# 前置条件（团队安装前必读）

本 skill 在审核流程中会调用**企业微信文档**读取法务背书词汇清单。新用户使用前需要完成以下配置：

## 1. 确认企微 CLI 已安装

运行以下命令确认：
```bash
which wecom-cli && wecom-cli --version
```
如未安装，联系锴哥获取安装包。

## 2. 确认企微机器人已授权「文档」权限

运行测试命令：
```bash
wecom-cli doc create_doc '{"doc_type": 3, "doc_name": "测试文档"}'
```
如返回 `errcode: 851014`（authorization expired），需重新授权：
> 企业微信 → 工作台 → 智能机器人 → 找到对应机器人 → 授权管理 → 开启「文档」权限

## 3. 确认企微机器人已授权「获取成员文档内容」权限

运行测试命令：
```bash
wecom-cli doc get_doc_content '{"docid":"<某个docid>","type":2}'
```
如返回 `errcode: 851008`（partial no authorization），需再次授权，勾选「获取成员文档内容」。

> ⚠️ **团队成员协作说明**：法务背书词汇清单托管在企业微信文档，**所有团队成员均可直接编辑**，无需联系锴哥手动录入。
