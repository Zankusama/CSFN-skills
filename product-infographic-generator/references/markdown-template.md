# 信息图中间文件模板（Markdown）

> 作用：生成 HTML 前的中间产物。**严格按类型选用对应模板**，禁止跨类型混写。
> 所有内容必须来自 brief，不允许编造；brief 缺失写 `[待补充]`。

---

## 模板 A：产品一页纸

文件名建议：`{产品名}_信息图内容.md`

```markdown
# {产品名} · 产品一页纸信息图内容

## 一、基础信息
- 产品名称：{产品名}
- 口号：{产品口号}
- 口播文案：{口播文案}

## 二、核心卖点 POD（4个）
### POD1：{标题}
- 特性：{...}   <!-- 标签名严格用「特性」非「特征」 -->
- 优势：{...}
- 利益：{...}   <!-- 利益用绿色标签样式 [xxx] 呈现 -->
- vs竞品：{...}（仅一句话参照，不展开竞品）
### POD2~4：同上结构

## 三、使用场景（N个）
- {场景1}：{描述}
- {场景2}：{描述}
- {场景3}：{描述}

## 四、产品系列（如有）
> 区分维度见 brief：香型 / 功效 / 联名款 / 规格 / 温度 等，名称动态
### {系列1名称}：{描述}
### {系列2名称}：{描述}
### {系列3名称}：{描述}（如更多按实际添加，0个则删除本板块）

## 五、产品信息
### 产品规格
- {规格条目}
### 使用方法
- {步骤条目}
### 适用人群
- {人群条目}
### 禁忌人群
- {禁忌条目}

## 六、注意事项（N条）
- {注意事项条目}

## 七、常见问题 FAQ（N条）
### Q1：{问题}
A：{答案，2句以内}
### Q2~N：同上

---
⛔ 禁止板块（本文件为「产品一页纸」类型）：
  禁止包含：竞品定位 / 竞品参数表 / 雷达图 / 打分数据 / 洞察提炼 / 购买理由
  （vs竞品仅允许作为 POD 内的简短参照句，不可独立成板块）
```

## 模板 B：竞品对比

文件名建议：`{产品名}_竞品对比内容.md`

```markdown
# {产品名} · 竞品对比信息图内容

## 一、竞品定位（N家，=brief竞品数）
> ⛔ 规则：每卡只写「品牌心智(消费者怎么认知它) + 产品特点(核心差异化)」，可选价格差异；
> 1句约20-35字，对标标杆图紧凑密度；❌禁止内部定位词(引流款/利润款/形象款等)、竞品销量/市占率数据。
### 蔡氏福宁
- {品牌心智}；{产品特点}（{可选价格差异}）
### {竞品1名称}
- {品牌心智}；{产品特点}
### {竞品2名称}
- {品牌心智}；{产品特点}
### {竞品3名称}
- {品牌心智}；{产品特点}

## 二、核心参数速查（按品牌独立呈现，不强制与蔡氏福宁一一映射）
> ⛔ 规则：每个品牌**独立列出** brief 中描述该品牌的参数/方面，不与蔡氏福宁或其他竞品强行对齐同一维度；
> 某品牌 brief 未描述的方面，该品牌小节下直接不写（不写[待补充]、不强行凑对称、也不砍蔡氏福宁独有项）；
> 每品牌 4-8 个要点，差异化值用**加粗**；这是"一眼看差异"的速查，不是完整规格表。
### 蔡氏福宁
- {方面1}：**{差异化值}**
- {方面2}：**{值}**
### {竞品1名称}
- {方面a}：{值}
### {竞品2名称}
- {方面b}：{值}
### {竞品3名称}
- {方面c}：{值}

## 三、雷达图后台数据（⚠️ 不呈现到 HTML）
| 维度 | 蔡氏福宁 | {竞品1} | {竞品2} | {竞品3} |
|:----|:--------|:-------|:-------|:-------|
| {维度1} | 3.5 | 3.5 | 3.5 | 1.0 |
| {维度2} | 3.5 | 2.5 | 3.0 | 1.0 |
| ... | ... | ... | ... | ... |
> 此表仅用于驱动雷达图形状，分数经用户确认。
> 分数不进入 HTML 任何可见位置（刻度/tooltip/卡片文字均禁数字）。

## 四、多维度综合对比洞察（数量 = 雷达维度数，定性，无数字，无依据行）
> 洞察卡数量 = 雷达维度数量，每张对应一个雷达维度；按雷达维度顺序左右均分（前半维度→左列、后半→右列），保证左右平衡；卡片品牌色/中性边框仍按该维度谁领先上色。圈号①②③由 HTML 生成。
### 洞察1：{维度名}
- 结论：{定性结论句，差异词用 **加粗**}
### 洞察2：{维度名}
- 结论：{...}
### 洞察3：{维度名}
- 结论：{...}
### 洞察4：{维度名}（如有）
- 结论：{...}
### 洞察5：{维度名}（如有）
- 结论：{...}
### 洞察6：{维度名}（如有）
- 结论：{...}

## 五、购买建议（📊 客观格局分析）
> 一段连贯的渠道向格局分析（3-5句）：总述竞争格局 → 分述各家核心长处（竞品A长于'xx'，竞品B长于'yy'）→ 聚焦蔡氏福宁差异化壁垒 → 适用人群建议。关键词用单引号'xxx'标注；禁止数字/星级/百分比；所有事实来自 brief。
{购买建议正文}

---
⛔ 禁止板块（本文件为「竞品对比」类型）：
  禁止包含：产品自身 POD 详述 / 产品系列卡片 / 使用场景 / 口播文案
  （产品信息仅允许作为「核心参数速查」表中的对比参照）
```

---

## 防跨类型污染自检（生成后执行）

- [ ] 一页纸 md 无「竞品定位/参数表/雷达图/打分/洞察提炼/购买理由」板块
- [ ] 竞品对比 md 无「产品 POD 详述/产品系列卡片/使用场景/口播文案」板块
- [ ] 两个 md 均含 `⛔ 禁止板块` 声明
- [ ] 竞品对比 md 的「雷达图后台数据」明确标注不进 HTML
